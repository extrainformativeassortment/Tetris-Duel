# Tetris-Duel
A Tetris game with single- and two-player (Tetris Battle-esque mode) modes built in WinForms app.

### Controls

**Single-Player Mode**

| Action      | Key       |
| :---------- | :-------- |
| Move Left   | `Left Arrow` |
| Move Right  | `Right Arrow`|
| Soft Drop   | `Down Arrow` |
| Hard Drop   | `Space`    |
| Rotate CW   | `Up Arrow` |
| Hold        | `Z`       |

---

**Two-Player Duel Mode**

**Player 1**

| Action      | Key       |
| :---------- | :-------- |
| Move Left   | `A`       |
| Move Right  | `D`       |
| Soft Drop   | `S`       |
| Hard Drop   | `Space`   |
| Rotate      | `W`       |
| Hold        | `Z`       |

**Player 2**

| Action      | Key           |
| :---------- | :------------ |
| Move Left   | `Left Arrow`  |
| Move Right  | `Right Arrow` |
| Soft Drop   | `Down Arrow`  |
| Hard Drop   | `Numpad 0`    |
| Rotate      | `Up Arrow`    |
| Hold        | `Numpad 1`    |
